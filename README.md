# Dotfiles

- **`home/`** - My dotfiles
- **`default/`** - System-level configurations
- **`install/`** - Installation scripts
- **`scripts/`** - Extra scripts

## Setup

```sh
git clone git@github.com:dougrocha/dotfiles.git
cd dotfiles

# Linux
./linux-setup.sh

# Macos
./stow
```
